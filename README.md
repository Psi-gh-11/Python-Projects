# CGPA & SGPA Calculator

A simple command-line calculator for computing **SGPA (Semester Grade Point Average)** and **CGPA (Cumulative Grade Point Average)** based on the **Indian grading system**.

## Features

- **Calculate SGPA**: Compute semester GPA using marks or grades
- **Calculate CGPA**: Calculate cumulative GPA across multiple semesters
- **Indian Grading Scale**: 
  - O (Outstanding): 90-100 → 10.0
  - A+ (Excellent): 85-89 → 9.0
  - A (Very Good): 80-84 → 8.0
  - B+ (Good): 75-79 → 7.0
  - B (Above Average): 70-74 → 6.0
  - C (Average): 60-69 → 5.0
  - D (Pass): 50-59 → 4.0
  - F (Fail): <50 → 0.0
- **Flexible Input**: Calculate using marks or grades
- **Grade Insights**: View grade ranges and score interpretations

## Quick Start

```bash
python cgpa-calculator.py
```

Follow the on-screen prompts to enter your semester details and calculate your GPA.
